# Previews 目录

此目录用于存放模板预览图（PNG 格式）。

## 命名规范

文件名与 `registry.json` 中的模板 `name` 字段一致：

- `academic.png`
- `academic_conference.png`
- `business.png`
- `gaokao_review.png`
- `minimal_clean.png`
- `product_launch.png`
- `project_report.png`
- `startup_pitch.png`
- `teaching.png`
- `thesis_defense.png`
- `training_workshop.png`
- `year_end_review.png`

## 规格建议

- **比例**：16:9（标准宽屏演示比例）
- **分辨率**：1920×1080 或 1280×720
- **格式**：PNG（优化后）
- **文件大小**：单个不超过 200KB
- **内容**：封面页 + 1-2 页内页的拼接预览

## 生成方式

可通过 pro-ppt-gen 技能的渲染管线将模板 PPTX 转为全页 PNG 预览后放置于此。
